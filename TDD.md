# SleepLate (护眼小卫士) 测试设计文档 (TDD)

## 1. 测试策略

### 1.1 测试原则
- **分层测试**: UI层、业务层、数据层独立测试
- **UI还原验证**: 验证界面元素与设计稿完全一致
- **功能覆盖**: 核心功能高覆盖，边界条件全覆盖

### 1.2 测试类型
1. **单元测试**: 各模块独立功能
2. **集成测试**: 多模块协作
3. **UI测试**: 验证界面布局、颜色、文字
4. **端到端测试**: 完整用户操作流程

## 4. 集成测试用例

### 4.1 完整计时流程

#### TC_INT_001: 正常工作流程
```
步骤:
1. 启动应用，加载配置
2. 设置工作45分钟，休息5分钟
3. 点击开始计时
4. 等待45分钟倒计时结束
5. 观察全屏锁定提醒
6. 5分钟后自动解锁

验证点:
- 计时器正常运行
- 倒计时准确（误差<1秒）
- 提醒按时触发
- 统计正确记录
```

#### TC_INT_002: 暂停后继续
```
步骤:
1. 开始计时
2. 等待10分钟
3. 暂停
4. 等待5分钟
5. 恢复计时
6. 验证剩余时间正确

验证点: 暂停时间不计入倒计时
```

#### TC_INT_003: 取消计时
```
步骤:
1. 开始计时
2. 点击取消
3. 验证计时器停止

验证点: 倒计时归零，无提醒触发
```
