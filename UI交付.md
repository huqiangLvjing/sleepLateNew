# 护眼小卫士 界面UI布局&元素完整规范
## 一、全局通用设计规范
### 1. 整体风格
现代圆角卡片式UI，明亮商务简约风格，层次分明，柔和轻量化阴影，视觉干净清爽。

### 2. 全局统一样式
- 通用圆角：12px ~ 16px
- 卡片样式：纯白色背景 + 浅灰色柔和投影
- 页面全局背景色：#f5f7fa
- 字体：无衬线常规字体，标题加粗，正文常规字重

### 3. 全局标准色值
| 用途分类 | 十六进制色值 |
| ---- | ---- |
| 顶部导航渐变主色 | #1677ff 渐变至 #722ed1 |
| 成功状态绿色 | #52c41a |
| 警告提示黄色 | #faad14 |
| 危险操作红色 | #ff4d4f |
| 正文主要文字色 | #1f2329 |
| 辅助灰色文字色 | #86909c |
| 卡片纯白底色 | #ffffff |
| 温馨提示框底色 | 浅黄弱边框浅色底 |
| 验证页顶部背景 | 红色渐变配色 |

## 二、主界面 完整布局与元素
### 整体布局层级
顶部渐变导航栏
├─ 核心计时展示卡片
├─ 左右双列功能设置卡片
├─ 全局文字提示公告栏
└─ 底部版本信息 + 功能操作按钮

### 1. 顶部导航栏
- 背景：蓝紫色渐变
- 左侧：圆形时钟图标 + 主标题【护眼小卫士】+ 副标题【科学用眼 · 定时休息】
- 右侧：绿色圆角状态标签 文字：运行中

### 2. 核心计时卡片
- 布局居中，浅蓝底色
- 顶部居中标题：距离下次休息还有
- 倒计时区域：三组白色圆角数字输入框，格式 时 : 分 : 秒
- 数字下方分别标注：小时、分钟、秒
- 底部统计数据区
  - 左侧：今日已使用电脑 + 时长数据
  - 右侧：今日已休息 + 休息次数

### 3. 左侧时间段设置卡片
- 头部：日历图标 + 卡片标题 + 功能简介 + 右侧跳转箭头
- 内容区：绿色状态圆点 + 工作日时段、周末时段两行时间文本

### 4. 右侧高级设置卡片
- 头部：齿轮图标 + 卡片标题 + 功能简介 + 右侧跳转箭头
- 内容区：展示已配置参数
  - 休息时长数值
  - 工作时长数值
  - 退出密码保护开启状态

### 5. 公告提示栏
- 样式：黄色边框浅黄底色提示条
- 左侧感叹号图标，右侧固定提示文案
- 文案内容：本程序已启用防退出保护，退出程序需要输入管理员密码。请牢记您设置的密码，超级密码可用于紧急解锁。

### 6. 底部区域
- 左下角：浅灰色小字版本号
- 右下角：灰色最小化按钮、红色退出程序按钮

## 三、时间段设置页面 布局与元素
### 整体布局层级
顶部返回导航栏
├─ 功能温馨说明卡片
├─ 提醒时长设置卡片
├─ 生效时间段管理卡片
├─ 提醒方式选择卡片
└─ 底部取消、保存操作按钮

### 1. 顶部导航栏
- 左侧白色返回箭头
- 页面主标题 + 小号功能副标题

### 2. 温馨说明卡片
- 左侧灯泡图标 + 完整功能说明文字
- 作用说明：仅在设置时间段内休息提醒生效

### 3. 提醒间隔设置区域
- 分为左右两个设置项
  1. 连续工作时长：加减按钮 + 数值输入框 + 单位分钟 + 底部健康建议小字
  2. 每次休息时长：加减按钮 + 数值输入框 + 单位分钟 + 底部健康建议小字

### 4. 生效时间段管理
- 卡片标题 + 右侧蓝色【添加时段】按钮
- 时段列表单条结构
  - 左侧开启/关闭状态开关
  - 时段名称、星期重复选择区域
  - 开始时间选择框、结束时间选择框
  - 右侧编辑图标、删除图标

### 5. 提醒方式设置
- 单选选项组（默认选中第一项）
  1. 全屏锁定提醒 + 配套功能说明
  2. 弹窗提醒 + 配套功能说明
  3. 静默提醒 + 配套功能说明
- 独立开关项：提醒音效 开启/关闭

### 6. 底部按钮
左侧灰色取消按钮、右侧蓝色保存设置按钮

### 5. 底部操作按钮

## 六、AI界面复刻强制约束规则
1. 全局所有卡片圆角、阴影、内外边距严格按照文档统一尺寸，禁止自定义修改
2. 全部文字内容、文案语句一字不差原样还原
3. 文字颜色、字体层级、对齐方式严格遵循规范
4. 所有开关、输入框、加减控件、图标保持整套UI统一风格
5. 按钮配色、默认状态、选中状态、警告状态严格对应设计
6. 页面整体居中排版，界面比例、板块分布严格匹配布局结构
7. 交互控件样式、悬浮视觉效果统一整套设计语言
